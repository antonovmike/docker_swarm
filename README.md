# docker_swarm

The algorythm creates "config.yml" file

(Fix description)

To start docker:
```bash
sudo docker run -it --rm rust /bin/sh
```
or
```bash
sudo docker image ls
sudo docker run -it --rm IMAGE_NAME /bin/sh
```

Clone project
```bash
git clone https://github.com/antonovmike/docker_swarm.git
```

Start project
```bash
cd docker_swarm
cargo run
```
